📖 使用说明
实时录音：点击「开始录音」，允许麦克风后即可实时转写（需 Chrome / Edge）。
音频上传：上传 MP3/WAV/M4A 文件，配置 API Key 后自动调用云端转写。
标记重点：选中文字后点击「⭐ 标记重点」，内容前后加上【重点】标记。
标记待办：选中文字后点击「☑ 标记待办」，内容前加上 [待办] 标记。
保存会议：点击「💾 保存会议」，输入名称后保存到本地浏览器。
历史记录：点击顶部「历史记录」标签，查看、导出或删除已保存的会议。
