🔑 语音识别 API Key
⚡ Groq 免费 · 速度最快 每天 ~2 小时音频
1. 访问 console.groq.com 注册账号（支持 Google 登录）
2. 进入 API Keys → 点击 Create API Key → 复制 key
3. 将 key 粘贴到上方「API Key」输入框
4. Endpoint 填写：https://api.groq.com/openai/v1/audio/transcriptions
💡 免费层每天可转写约 7200 秒（≈ 2 小时）音频，超出后次日重置，足够日常会议使用。
🇨🇳 SiliconFlow 免费 · 中文效果好 注册送 ¥14 额度
1. 访问 siliconflow.cn 注册账号（支持手机号）
2. 进入「API 密钥」→ 点击「新建 API 密钥」→ 复制 key
3. 将 key 粘贴到上方「API Key」输入框
4. Endpoint 填写：https://api.siliconflow.cn/v1/audio/transcriptions
💡 使用 SenseVoiceSmall 模型，注册即送 ¥14 免费额度，按实际用量计费，中文识别准确率更高。
💡 Key 仅保存在本地浏览器，不上传到任何服务器。实时录音转写无需配置 Key。
📖 使用说明
实时录音：点击「开始录音」，允许麦克风后即可实时转写（需 Chrome / Edge）。
音频上传：上传音频文件，配置 API Key 后自动调用云端转写，并自动保存到历史记录。
标记重点：在笔记区选中文字后点击「⭐ 标记重点」，文字变为紫色高亮，并显示在左侧边栏。
标记待办：选中文字后点击「☑ 标记待办」，文字前加上复选框，并显示在左侧边栏。
保存会议：点击「💾 保存会议」，输入名称后保存到本地浏览器（不上传任何数据）。

⚠️ 若页面未更新，按 Ctrl+Shift+R 强制刷新。
